# Official SampleMod_NewCar Reference

This directory intentionally does not contain a copied SampleMod_NewCar.

Run:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/install_official_sample.ps1
```

The installer downloads the current official sample from:

https://github.com/BohemiaInteractive/Arma-Reforger-Samples/tree/main/SampleMod_NewCar

It is installed into the standard Reforger Workbench addons directory so Workbench can open and register it normally.
