INSTALLATION

Install the Blender addon

  1. Download reforger_vehicle_checker.zip.
  2. Open Blender and choose Edit > Preferences > Add-ons.
  3. Choose Install from Disk and select the ZIP.
  4. Enable Reforger Vehicle Checker.
  5. Press N in the 3D Viewport and open the Part Fixer or RVC tab.

WORKFLOW

Recommended order

Setup

Set asset name, export directory, wheelbase, and wheel radius.

Organize

Assign and review vehicle parts before finalizing.

Rig

Build bones, bind rigid parts, and place memory points.

Collide

Create simple convex physics and purposeful FireGeo.

Validate

Resolve blocking RVC errors.

Export

Use master, glass, wheel, and light FBX profiles.

COLLISION RULE

Main vehicle physics must be simple convex blocks.

Use several `UCX_MainCol_*` or `UBX_MainCol_*` hulls. Keep glass, lights, wheels, and wheel `VehicleComplex` in their own slot exports.
