INSTALLATION

From download to Blender in five steps

  1. Download reforger_vehicle_checker.zip.
  2. Open Blender and choose Edit > Preferences > Add-ons.
  3. Choose Install from Disk and select the ZIP.
  4. Enable Reforger Vehicle Checker.
  5. Press N in the 3D Viewport and open the Part Fixer or RVC tab.

THE RULE THAT MATTERS

A complete vehicle outline is usually not one valid UCX.

Hoods, windshields, roofs, turrets, and rear steps create a concave outline. Build several individually convex blocks that meet at the visible breakpoints. Use Convexify Selected UCX after manual point adjustments and Validate before export.
